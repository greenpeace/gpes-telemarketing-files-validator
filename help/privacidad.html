Privacidad y seguridad

<< Menu

Al usar el validador no estás subiendo ficheros a ningún servidor. Los datos nunca salen de tu ordenador.

La conexión con el servidor está encriptada (https) para garantir que estás usando el código que hemos desarrollado, sin interferencias.

Evita usar servicios como Google Docs o semejantes porque al hacerlo estás subiendo datos a servicios, lo que añade riesgos a la seguridad de los datos.
